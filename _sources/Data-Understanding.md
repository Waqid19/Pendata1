# Data-Understanding
Data Understanding adalah tahap dalam proses analisis data (seperti CRISP-DM) yang berfokus pada memahami data yang akan digunakan sebelum dilakukan pengolahan atau pemodelan.

Tahap ini dilakukan setelah Business Understanding, dan bertujuan memastikan data yang tersedia relevan, berkualitas, dan sesuai dengan tujuan bisnis.

## Tujuan Data Understanding
- Mengenali struktur dan isi data
- engidentifikasi masalah kualitas data
- Menemukan pola awal atau insight awal

## Langkah-Langkah dalam Data Understanding
1. Mengumpulkan Data (Collect Initial Data)
Mengambil data dari berbagai sumber seperti:
- Database perusahaan
- File Excel/CSV
- API
- Sistem internal

2. Mendeskripsikan Data (Describe Data)
Memahami:
- Jumlah baris dan kolom
- Tipe data (numerik, kategorikal, tanggal, dll.)
- Sumber data

3. Mengeksplorasi Data (Explore Data)

Melakukan analisis awal seperti:
- Statistik deskriptif (mean, median, min, max)
- Distribusi data
- Korelasi antar variabel
- Visualisasi (grafik, histogram, dll.)

4. Memverifikasi Kualitas Data (Verify Data Quality)

Mengecek:
- Missing value (data kosong)
- Duplikasi data
- Outlier
- Inkonsistensi format
