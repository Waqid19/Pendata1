# Data-Preparation
Data Preparation adalah tahap dalam proses analisis data (seperti CRISP-DM) yang berfokus pada menyiapkan data agar siap digunakan untuk proses pemodelan atau analisis lebih lanjut.

Tahap ini biasanya memakan waktu paling lama karena data mentah sering kali belum bersih dan belum terstruktur dengan baik.

## Tujuan Data Preparation
- Membersihkan data agar bebas dari kesalahan
- Mengubah format data agar sesuai untuk analisis
- Menggabungkan berbagai sumber data
- Memilih variabel yang relevan

## Langkah-Langkah dalam Data Preparation
1. Data Cleaning (Pembersihan Data)
Meliputi:
- Menghapus atau mengisi missing value
- Menghapus duplikasi data
- Menangani outlier
- Memperbaiki kesalahan penulisan atau format

2. Data Integration (Integrasi Data
Menggabungkan data dari beberapa sumber menjadi satu dataset yang konsisten.
Contoh:
- Menggabungkan data pelanggan dengan data transaksi.

3. Data Transformation (Transformasi Data)
Mengubah data agar sesuai untuk analisis:
- Normalisasi atau standarisasi
- Encoding data kategorikal (misalnya jenis kelamin menjadi 0 dan 1)
- Mengubah format tanggal
- Membuat fitur baru (feature engineering)

4. Data Reduction (Reduksi Data)
Mengurangi kompleksitas data:
- Memilih variabel penting saja
- Menghapus kolom yang tidak relevan
- Menggunakan teknik seperti PCA (jika diperlukan)
